Language Translator

Translates text of one language to another.

Scroll
Project Date October 7, 2020
Role Web Development, APIs
Client Personal
Visit https://languageinterpreter.herokuapp.com

This is a language translator which allows you to translate text of one language to another. Enter some text in the text area, select the languages and boom! the text is translated. Right now it has 13 language options.

This application uses IBM Watson at the back-end for translation and is built using Django.
